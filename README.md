# EG Reveal Hero

Интерактивная главная страница с эффектом проявления второго изображения под курсором. Поддерживает мышь и касания на мобильных устройствах.

## Просмотр

После первой успешной публикации сайт будет доступен по адресу:

**https://axdsgn-create.github.io/eg-reveal-hero/**

## Локальный запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

## Изображения

- `assets/base.jpg` — постоянный слой.
- `assets/reveal.jpg` — слой, проявляющийся под курсором.

Изображения должны иметь одинаковые размеры и совпадающую композицию. Размер маски задаётся переменной `--mask-size` в `styles.css`.
